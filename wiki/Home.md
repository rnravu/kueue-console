# Welcome to the Kueue Console Wiki

This is the home page for the Kueue Console documentation. Kueue Console provides a web interface for interacting with and managing Kubernetes Kueue workloads.

## Getting Started

- **[Installation & Setup](Installation)**
- **[Architecture & Design](Architecture)**
- **[Configuration](Configuration)**

## Features

- **Dashboard**: High-level overview of Cluster Queues, Local Queues, and Workloads.
- **Job Templates**: Submit workloads via pre-configured Job Templates.
- **Monitoring**: Real-time updates and diagnostic logs for Jobs and Queues.
